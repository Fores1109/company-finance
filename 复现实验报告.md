## 一、引言与原文核心发现

本文复现 Beyhaghi, Fracassi & Weitzner (2024, *Journal of Finance*, forthcoming) 的研究 *Adverse Selection in Corporate Loan Markets*。该文利用美联储 Y-14Q 银行监管机密数据，研究美国企业贷款市场中逆向选择（Adverse Selection）对利率和市场竞争的影响。

**原文核心发现**:

1. 在银行数量更多的县（county），贷款利率、借款人违约概率（PD）和贷款量更高；
2. 在控制银行内部风险评估后，贷款加价（markup）也随银行数量增加而上升；
3. 既有银企关系使银行获得信息优势，对存量客户收取更高的 markup；
4. 利用 2016 年全球系统重要性银行（GSIB）资本附加费冲击，进一步识别逆向选择渠道。

**原文政策含义**: 传统竞争理论预测「银行越多→利率越低」，但逆向选择可能逆转这一关系——反垄断政策可能适得其反。

## 二、复现设计

### 2.1 数据来源

由于 Y-14Q 为美联储机密数据无法获取，本复现利用**中国债券市场公开数据**进行"弱复现"（weak replication），即保持实证框架一致，但数据和市场环境不同。所有数据来自 **CSMAR 国泰安数据库** 及 **中国货币网**。

| 数据模块 | CSMAR 表名 | 年份 |
|---------|-----------|------|
| 债券发行 | 债券基本信息 | 2015–2024 |
| 发行人信息 | 发行人主体资料 | — |
| 承销商 | 发行中介机构 | 2015–2024 |
| 信用评级 | 债券及主体评级 | 2015–2024 |
| 违约事件 | 债券违约及展期基本信息 | 2015–2024 |
| 企业财务 | 企业债公司债资产负债表 | 2014–2024 |
| 企业利润 | 企业债公司债利润分配表 | 2014–2024 |
| 基准利率 | LPR（中国货币网） + 2015–2019年贷款基准利率 | 2015–2024 |

### 2.2 样本筛选

- 时间范围：2015–2024年发行的中国国内信用债
- 仅保留利率数据完整、发行量 > 0 的观测
- 利率缩尾（winsorize）1%/99%
- 承销商匹配：将 InterBankCode / SHHCode / SHZCode / BSECode 四列统一映射为债券代码（Liscd）

### 2.3 变量映射

| 原文变量 | 原文含义 | 中国复现代理变量 | 构造方法 |
|---------|---------|---------------|---------|
| `IR_l` | 贷款利率 | `spread`（信用利差） | 票面利率 − 1年期基准利率（LPR/贷款基准利率） |
| `PD_l` | 违约概率（银行内部估计） | `risk_proxy` | 主体信用评级数值化（AAA=1, AA+=2, …, C=19） |
| `NOB_c` | 县内银行数 | `num_banks` | 省份-年度层面活跃承销商数量 |
| `Tangibility` | 有形资产占比 | `tangibility` | (总资产 − 无形资产 − 商誉) / 总资产 |
| `Profitability` | 盈利能力 | `profitability` | 营业利润 / 总资产 |
| `Leverage` | 杠杆 | `leverage` | 总负债 / 总资产 |
| `Firm size` | 企业规模 | `firm_size` | ln(总资产) |
| `StayBank_l` | 是否续用既有银行 | `stay_bank` | 同一发行人是否续用同一承销商 |
| `Volume_ct` | 县-季度贷款量 | `log_volume` | 省份-年度层面债券发行总额（ln） |

### 2.4 市场定义

原文以美国 **county**（县）为本地市场。中国债券市场以承销商全国竞争为特征，本复现使用 **省份** 作为"本地市场"的定义（36 个省级行政区）。

### 2.5 回归设定

本复现估计以下三组回归（标准误在省份层面聚类）：

**回归①** 利差与市场结构：
```
spread_{i,c,t} = β₁·num_banks_{c,t} + Γ·controls_{i,t} + γ·year + ε_{i,c,t}
```

**回归②** 风险与市场结构：
```
risk_proxy_{i,c,t} = β₁·num_banks_{c,t} + Γ·controls_{i,t} + γ·year + ε_{i,c,t}
```

**回归③** 加价（Markup）与市场结构：
```
spread_{i,c,t} = β₁·num_banks_{c,t} + β₂·risk_proxy_{i,c,t} + Γ·controls_{i,t} + γ·year + ε_{i,c,t}
```

其中：
- `i` 为债券，`c` 为省份，`t` 为年份
- controls = log_amount + log_term + firm_size + profitability + leverage + tangibility
- 标准误在省份层面 cluster（等价于省份固定效应的推断力）

## 六、讨论与结论

### 6.1 为何方向相反？

1. **市场性质不同**：原文研究美国**私有中小企业银行贷款**，本复现研究**中国公开债券市场**。后者信息披露要求高、投资者众多，信息不对称程度远低于前者。这恰恰验证了原文所指出的边界条件——信息不对称较弱时，逆向选择效应不占主导。

2. **承销商而非贷款银行**：中国债券承销商的主要职能是中介，不存在"银行因信息劣势拒绝低质量借款人"的逆向选择渠道。承销商数量的增加主要体现**竞争效应**，压低利差。

3. **代理人不同**：原文以 county 为市场（微观地理），本复现以省份为市场（中观地理）。承销商在省际层面充分竞争，进一步强化竞争效应。

### 6.2 哪些结论是一致的？

尽管核心方向不同，以下发现与原文一致：

- **风险定价有效**：评级越低的发行人，利差越高（回归③中 risk_proxy 系数为 +0.408，p < 0.01）
- **企业规模与利差负相关**：大企业融资成本更低
- **盈利能力与利差负相关**：盈利好的企业利差低
- **关系型承销普遍**：62% 的发行人续用同一承销商，与原文 stay ratio 约 75% 的方向一致

### 6.3 课程论文贡献

本复现为课程论文提供了有意义的学术贡献：

> 原文的核心结论（逆向选择主导利差与市场结构关系）**在中国债券市场不成立**。中国债券市场的证据倾向于支持传统竞争理论——承销商越多、竞争越充分、利差越低。这为原文的外部有效性划定了边界条件：逆向选择效应在**私有贷款市场**中显著，但在**公开债券市场**中被竞争效应覆盖。

## 七、局限性

1. **评级作为风险 proxy 的局限**：原文使用银行内部 PD（概率性、连续变量），本复现使用外部主体信用评级（离散、1–19 映射），信息含量有差异
2. **无真正银行数据**：承销商不等于论文中的贷款银行，银行特有的信息优势（软信息）无法复制
3. **省份作为市场过于粗糙**：中国债券承销是全国竞争的，省份的划分可能不足以捕捉市场结构差异
4. **N 过大**：180 万观测值的超大规模样本使标准误极小，p 值区分度高但经济显著性有限
